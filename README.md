# 💎 GIOIA - Elite QR Digital Menu Engine (V8.0)

A high-end, high-performance digital menu solution designed for Luxury Restaurants, Lounges, and Boutique Hotels. Built with a focus on "Mobile-First" experience and "Tesla-inspired" fluid animations.

[[Live Demo Link Here](https://lhnd12.github.io/GioiaMenuQR/main.html)]

## ✨ Key Features

- **Luxury Aesthetic:** Deep Emerald & Gold theme with custom minimalist SVG iconography.
- **Smart Search & Auto-Scroll:** Instant filtering with a "Press Enter to Scroll" feature that takes the guest directly to the dish.
- **Tesla-Style Loader:** A sophisticated entrance animation that sets the tone for a premium dining experience.
- **Zero-Icon Modal:** A clean, distraction-free viewing mode that puts 100% focus on high-quality food photography.
- **Optimized Performance:** Lightweight vanilla JavaScript (No heavy frameworks) for lightning-fast loading on 3G/4G networks.
- **Category Counters:** Dynamic counters that show guests exactly how many options are available in each section.

---

## 🛠️ Technical Stack

- **Frontend:** HTML5, CSS3 (Custom Variables & Keyframes)
- **Engine:** Vanilla JavaScript (ES6+)
- **Icons:** Custom stroke-based SVGs (Uniform Design System)
- **Deployment:** Optimized for GitHub Pages / Netlify / Vercel

---

## 📱 User Experience (UX)

1. **The Entrance:** Guest scans QR -> Luxury Loader appears -> Menu reveals with a smooth stagger effect.
2. **The Discovery:** Guest can filter by category (Pizza, Salads, etc.) or use the global search.
3. **The Selection:** Pressing "Enter" on search automatically scrolls to and highlights the best match.
4. **The Detail:** Clicking an item opens a sleek modal showing the dish in full glory without UI clutter.
